# Your agentsfs

This folder is a memory that you and your AI agents share. Everything in it is a plain file you can open, edit, move, or delete with any tool you like — the agents follow the rules in [AGENTS.md](AGENTS.md), and you don't have to follow any rules at all.

It's a git repository, so every change is recorded and reversible, and you can sync it anywhere git goes. To take your data elsewhere: copy the folder. That's it.
