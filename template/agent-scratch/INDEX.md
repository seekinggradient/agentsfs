# agent-scratch (ephemeral workspace)

This directory is the instance's **scratch space** — the role is set by the `agentsfs_role: scratch` marker above, not by the folder name. `agent-scratch/` is only the default.

Mess is legal here. Move anything worth keeping out before you finish, and describe it where it lands.
